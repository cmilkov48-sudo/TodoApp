# Todo App

## Описание на проекта
Проектът представлява трислойно конзолно приложение за управление на задачи (To-Do система).  
Потребителите могат да се регистрират, да влизат в системата и да управляват своите задачи.

Приложението реализира основните CRUD операции (Create, Read, Update, Delete) върху два типа данни:
- Потребители (Users)
- Задачи (Tasks)

---

## Архитектура

Проектът е реализиран като трислойно приложение:

### 🔹 Слой за данни (TodoApp.Data)
- Съдържа моделите (User, TaskItem)
- Съдържа AppDbContext
- Работи с база данни чрез Entity Framework Core

### 🔹 Слой за услуги (TodoApp.Services)
- Съдържа бизнес логиката
- UserService – регистрация и вход на потребители
- TaskService – управление на задачи

### 🔹 Презентационен слой (TodoApp)
- Конзолен интерфейс
- Взаимодейства с потребителя
- Извиква методите от service слоя

## Функционалности

### 👤 Потребители
- Регистрация
- Вход (Login)

### 📋 Задачи
- Добавяне на задача
- Преглед на всички задачи
- Преглед на завършени задачи
- Преглед на незавършени задачи
- Маркиране на задача като завършена
- Редактиране на задача
- Изтриване на задача

---

## Как да стартираме проекта

1. Отворете проекта във Visual Studio
2. Стартирайте приложението (Ctrl + F5)
3. Създайте потребител (Register)
4. Влезте в системата (Login)
5. Започнете да добавяте и управлявате задачи

Базата данни се създава чрез migrations.
